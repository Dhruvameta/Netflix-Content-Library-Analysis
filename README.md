# Netflix-Content-Library-Analysis

## Overview

This project performs Exploratory Data Analysis (EDA) on the Netflix Titles dataset using Python and Pandas.

The goal of this project is to explore Netflix's content library and answer business-oriented questions through data analysis. The project covers data loading, cleaning, filtering, aggregation, sorting, and text analysis using real-world data.

---

## Dataset

The dataset contains information about Netflix Movies and TV Shows, including:

* Title
* Type (Movie / TV Show)
* Director
* Country
* Release Year
* Rating
* Duration
* Genres
* Date Added

Dataset Source: Netflix Titles Dataset

## Analysis Performed

Some of the questions explored in this project include:

* How many Movies and TV Shows are available?
* Which countries produce the most Netflix content?
* Which year had the highest number of releases?
* What are the most common content ratings?
* How many missing values exist in the dataset?
* Which titles contain specific keywords such as "Love" or "King"?
* What trends can be observed in Netflix content over time?

---

## Key Findings

* Movies appear more frequently than TV Shows.
* A small number of countries contribute a large portion of Netflix content.
* Content releases increased significantly in recent years.
* Several columns contain missing values and require cleaning before advanced analysis.
